# LineBot
## How to use
1. Install Python Package
  > pip install -r requirements.txt
2. Run ngrok
  > uvicorn sport:app --reload
3. Test LineBot 
  > python main.py 
## Indtroduction
Calculator
- operator
  - `+`
  - `-`
  - `*`
  - `/`
- input (a,b can be negative)
  - `a+b`
  - `a + b`
  - `a+b =` 
- output
  - `calculation result` 
  - If you enter illegal characters or don't match the rules you will get the picture
